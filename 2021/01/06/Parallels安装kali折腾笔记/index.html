Parallels安装kali折腾笔记

安装版本：

kali-linux-2020.2-installer-amd64.iso
Parallels 16.0.1

Change Aliyun source

nano /etc/apt/sources.list 

#deb https://mirrors.aliyun.com/kali kali-rolling main non-free contrib
#deb-src https://mirrors.aliyun.com/kali kali-rolling main non-free contrib

Update Soft

apt-get clean
apt-get update
apt-get upgrade -y
apt-get dist-upgrade -y
apt-get install dkms kpartx printer-driver-postscript-hp

Install Parallels Tools

sudo -s
eject /dev/cdrom 
然后挂载tools
mkdir /media/pt
mkdir /root/Desktop/pt
mount /dev/cdrom /media/pt
cp -r /media/pt/  /root/Desktop/pt
cd /root/Desktop/pt/pt
./install

无法安装linux-headers


接下来的错误都是要查看日志文件了

cat /var/log/parallels-tools-install.log
如果是无法安装linux-headers的话，就要手动安装。
 先查看内核版本

uname -a
然后来这里http://http.kali.org/kali/pool/main/l/linux/下载三个对应内核版本的安装包手动安装
国内使用aliyun下载更快 https://mirrors.aliyun.com/kali/pool/main/l/linux/

linux-kbuild: linux-kbuild-xxxx_amd64.deb
linux-header-common: linux-headers-xxxx-common_xxxx_amd64.deb
linux-compiler-gcc: linux-compiler-gcc-xxx-amd64.deb
linux-headers: linux-headers-xxxx_amd64.deb
下载完成后，用dpkg命令安装deb包。
dpkg -i xxxxx.deb

装完Tools后登陆白屏问题解决
